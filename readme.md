# Free The Barber
A simple Django Todo List

# Initial Goal
Showcase Django skills
- CRUD functionality
- Use of models
- Use of forms
- Use of django templates
- Use of django urls
- Use of django views

# Tecnhologies used
* Django
* Crispy Forms
* Bootstrap 5

# Functionalities
* Create/List/Edit/Delete Todos

# How to Run
1. Download project
2. Install requirements from requirements.txt
3. `cd todo_app`
4. `python manage.py runserver`
